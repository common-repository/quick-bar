== Description ==

A quicklook into Quick Bar - Popup Notification Sidebar

    ★ Create a responsive floating sidebar or popup notice bar
    ★ Full control on popup sidebar content
    ★ Standard WordPress content editor
    ★ Timeout based popup sidebar display
    ★ Display based on browsed number of pages
    ★ Css settings like color, z-index, border etc
    ★ Shortcode for displaying popup notification sidebar in specific pages
    ★ Display specifically in pages/posts/home page

This plugin allows you to create a  popup notification sidebar with custom content in your site. You can place it like a floating sidebar on any side of the page using the available predefined positions.

You can customize the  popup notification sidebar display by configuring various configurations such as display logic settings ( based on timeout after page load, based on number of pages browsed,  repeat interval) and style settings(z-index, color, border etc). The plugin supports automatic and manual (shortcode) display.

You can use the popup notification sidebar to display any type of contents such as special promotions, subscription forms, social media icons, feedback forms and much more.

= 1. What is a "quick bar" or "sidebar" ? =

A quick bar  or sidebar is a floating dhtml popup bar used to display any content within a webpage, above the the page content. Sidebars are normally used to show social media links, subscription forms, contact forms, promotions, notifications etc

= 2. The Quick Bar - Popup Notification Sidebar plugin is not working properly =

Please check the wordpress version you are using. Make sure it meets the minimum version recommended by us. Make sure all files of the `Quick Bar - Popup Notification Sidebar` plugin are uploaded to the folder `wp-content/plugins/`

= 3. What kind of contents can be placed in a Quick Bar - Popup Notification Sidebar ? =

You may  use the Quick Bar - Popup Notification Sidebar to display any type of contents such as social media icons, special promotions, subscription forms, contact forms, notifications  and much more.

= 4. Can I display Quick Bar - Popup Notification Sidebar popup in specific pages only ? =

Yes, you can do this by using the shortcode provided from settings page.

= 5. Is the Quick Bar resposive ? =

Yes you can configure responsive popup by specifying height and width in percentage. You need to ensure from your end that the content inside the sidebar is also responsive.

= 6. Does it work with caching plugins ? =

You can turn on compatibility with cache plugins from settings page. Please  note that since this method uses ajax, the popup will be a rendered a bit slower than in normal mode.

= 7. Can I create multiple Quick Bar - Popup Notification Sidebar popup boxes  for different pages ? =

To create multiple quickbars (sidebars), you need to purchase the premium version.
